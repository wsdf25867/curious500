### 아이디어

* [궁금하면 500원](https://namu.wiki/w/%EA%B1%B0%EC%A7%80%EC%9D%98%20%ED%92%88%EA%B2%A9) 에서 아이디어를 얻은 질문 & 답변 게시판 서비스 토이프로젝트
* 유저마다 wan 이라는 재화가 있다.
* 질문을 하면 500이 차감된다.
* 답변을 일정량 벌고, 채택 된 답변은 500을 받는다.
